# /caveman

Slash entry point for the `caveman` skill. Activates persistent compressed-output mode.

## Arguments

`$ARGUMENTS`

## Delegation

Apply the `caveman` skill (`skills/caveman/SKILL.md`). Once active, all subsequent responses use caveman compression until the user says "stop caveman" or "normal mode".
